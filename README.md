# Hackathon 1: Laufzeitkomplexität von Sortierverfahren (3 Punkte)
Bitte das Repository clonen oder die Dateien runterladen.

my_bench.py ausführen.

1) Erweitern Sie my_bench.py um Bubblesort und messen Sie die Anzahl Vergleiche und Laufzeit für sinnvolle Anzahlen von N. (1 Punkt)

2) Erweitern Sie my_bench.py um die Algorithmen Bubblesort, Insertionsort, Selectionsort und Quicksort (optional: Mergesort)
und messen Sie die Anzahl Vergleiche und Laufzeit für sinnvolle Anzahlen von N. (1 Punkt)

3) Erweitern Sie die Messung um (vor-) sortierte Daten um best bzw. worst cases zu untersuchen, wo angemessen

4) Erweitern Sie my_bench.py um eine angemessen "schöne" grafische Darstellung der Ergebnisse. Verwenden Sie dazu entweder matplotlib oder erzeugen Sie eine csv-Datei und erzeugen die Grafik anders, z.B. mit Excel. Alternativ: übersichtliche "schöne" Tabelle
